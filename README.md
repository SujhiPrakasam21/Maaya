# MERN Login System

This is a simple MERN stack login and registration system.

## Technologies Used
- React.js
- Node.js
- Express.js
- MongoDB
- bcrypt for password hashing

## Features
- User Registration
- User Login
- Password Encryption
- MongoDB Database Storage
